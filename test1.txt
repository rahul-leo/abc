

# 1. index.html

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Simple Frontend</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <div class="container">
        <h1>Welcome to My Frontend App</h1>
        <p>This app is running inside Docker 🚀</p>
        <button onclick="showMessage()">Click Me</button>
        <h2 id="message"></h2>
    </div>

    <script src="script.js"></script>
</body>
</html>
```

# 3. script.js

```javascript
function showMessage() {
    document.getElementById("message").innerText = "Hello from Docker Frontend!";
}
```

---

# 4. Dockerfile

```dockerfile
# Use nginx image
FROM nginx:alpine

# Copy frontend files to nginx folder
COPY . /usr/share/nginx/html

# Copy nginx config
COPY nginx.conf /etc/nginx/conf.d/default.conf

# Expose port
EXPOSE 80

# Start nginx
CMD ["nginx", "-g", "daemon off;"]
```

---

# 5. nginx.conf

```nginx
server {
    listen 80;

    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
}
```

---

# Run the Project

## Step 1: Build Docker Image

```bash
docker build -t simple-frontend .
```

## Step 2: Run Docker Container

```bash
docker run -d -p 8080:80 simple-frontend
```
